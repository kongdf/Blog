# 年

* 生肖鼠:免年，龙年，猴年，鸡年，牛年。
* 生肖牛:虎年，猴年，鸡年，蛇年，鼠年。
* 生肖虎:牛年，马年，羊年，狗年，猪年。
* 生肖兔:鼠年，马年，猪年，羊年，狗年。
* 生肖龙:蛇年，猪年，鼠年，猴年，鸡年。
* 生肖蛇:龙年，狗年，鸡年，牛年，猴年。
* 生肖马:兔年，鸡年，虎年，狗年，羊年。
* 生肖羊:猴年，虎年，兔年，猪年，马年。
* 生肖猴:牛年，羊年，鼠年，龙年，年。
* 生肖鸡:鼠年，牛年，马年，年，龙年。
* 生肖狗:兔年，猪年，虎年，马年，年。
* 生肖猪:狗年，龙年，兔年，羊年，虎年。


一般来说，能有条件的尽量选择大利年来结婚，如果没有大利年，就选择不要相冲的年份（相冲以女命生肖，男命生肖，以女命为主，男命尽量避免
